PINNACLE's documentation
===================================

This is the documentation for PINNACLE (Publications from an Institute: Numbers, Networks, Authors and Citations Looking for Excellence.)

It aims at exploring the publication metrics, relations and networks
for researchers at an institute, in order to design strategies to
increase impact and productivity.
